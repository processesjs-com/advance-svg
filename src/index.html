Advance SVG Specification and Demo

Introduction

Advance SVG is a technique for publishing interactive graphics on internet. The graphics can describe any matter e.g., business processes, mind charts, technical manuals – anything that is worth to be described with a simple, at-a-glance visual presentation.

Advance SVG utilises the standard modern browsers’ abilities to draw high quality, scalable graphics (ref HTML5 SVG specification) and to take actions via JavaScript programming language.

The graphic below demonstrates Advance SVG.

This is Advance SVG insertion point

The technique involves three elements:

  1. A web page where graphic(s) will be embedded - this page for example.
  2. A JavaScript library - ASVG Script - availbale from various sources i.e. CDN and GitHub.
  3. Marked-up SVG graphic - the graphic above for example.

ASVG Script works at the background and monitors the page content for mark-ups, then it enhances the graphics on the fly. It adds: responsive behaviour, pop-up shapes, in-page links and icons. These features are explained in detail further down.

How it wors?

Advance SVG is not an authoring tool, hence it does not deal with neither creation of web pages nor with creation of graphics. Various tools can be used for these. In order to work, it is necessary to ensure that: first, the ASVG Script is included into the web page; second, the shapes are marked-up and finally, the web page includes insertion point where the graphic(s) will display.

The workflow below shows how Advance SVG works.

This is Advance SVG insertion point

The workflow steps are as follows:

  1. Create web page – this is outside of the Advance SVG scope. Can use any authoring tool.
  2. Add ASVG Script – it’s as simple as adding the following snippet into the HTML code
                    
                  <script src="https://d2a8hhqmsel69m.cloudfront.net/bundle.js"></script>
    Please note that adding the library to a Content Management System (CMS) like WordPress or SharePoint requires System Administrator’s work.
  3. Add insertion point - add the following snippet in the place where the graphic is required:
                    
                  <div data-asvg="page1.svg" data-asvg-show="page1.svg" style="width:100%;"> </div>
  4. Draw a graphic – this is outside of the Advance SVG scope. Can use any authoring tool.
  5. Mark up shapes – add data attributes to groups (g tags) in the SVG code. Here is a simple example of popuplink and the associated popup:
                    
                  <g data-asvg-popuplink=”1” …>
                    …
                  </g>
                  <g data-asvg-popup=”1” …>
                    …
                  </g>
  6. Provide SVG file - the marked up SVG file will be loaded dynamically by the ASVG Script and placed into the insertion point (see step 3).
  7. Enjoy ASVG - explore the examples on this page.
  8. Using MS Visio – MS Visio is a great authoring tool that allows adding custom data to the graphics. ASVG Script has a filter that can recognise that data and mark-up on-the-fly. Graphics created in MS Visio can be exported to SVG format and published directly. Tutorial about using MS Visio to create Advance SVG graphics will be provided separately.

Technical Specification

The table below explains the data attributes that can be added to shapes (SVG g tags).

Parameter name Value(s) Description SVG attribute
display L (large),
M (medium),
S (small) or
T (thumbnail)		 Provides responsive behaviour by switching between different 'scenes', depending on the graphic container's size. data-asvg-display
popup id Popup shape to be linked to a popuplink data-asvg-popup
popuplink id When selected (click or tap), it shows the associated Popup shape. data-asvg-popuplink
pagelink SVG_file_name Replace the current graphic with another SVG file. data-asvg-pagelink
icon icon_name position Currently, the icons that can be added are:
  • asvg-popup-close
  • asvg-popuplink-icon
  • asvg-pagelink-icon
The positions are: tl, tc, tr, ml, mc, mr, bl, bc and br for Top-Left … Top-Center … Top-Right and so on. 		data-asvg-icon
hreftarget e.g. _blank This works with hyperlinks for setting the target parameter - see HTML specification about the hyperlink targets. data-asvg-hreftarget
